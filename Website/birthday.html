Birthday command

Set your birthday or age, or view your own or another member's birthday or wish list

.birthday |{action}|

Alias: .bday

{action} - optional

If a search term (such as member's username, ID) is provided, the bot will pull up a list of matching members for you to view their birthday and wish list. If omitted, the homepage selector with available options will be brought up

If a number (under 10,000) is provided, the bot will ask you if you would like to set that as your birthday

If a date is provided, the bot will ask you if you would like to set that as your age

If {action} is omitted, the bot will pull up your birthday information, along with a listing of upcoming birthdays across the current server and anyone in a server you're in

Examples

.birthday

View your birthday, age, and upcoming birthdays

.bday september 21

Set the 21st night of September as your birthday

.bday bsl_dragonking

View birthday information of the user named BSL_DragonKing
