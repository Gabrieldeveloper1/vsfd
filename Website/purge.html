purge command

Purge messages

Quick purge: .purge [number]

Example - purge 10 posts: .purge 10

Advanced purge: use filters

filter_name:value

Multi-lining filters is optional. If you use filters, you must include 'purge:true' to actually purge for safety reasons. Separate different filters by a space or newline; separate filter:value with only a colon - no space. Use as many filters as necessary. If a value requires a space for multiple words, surround filter:value with quotes, like this: "filter:value". For the True/False values, you can set them as false to ignore them, but omitting them is also valid - all filters are optional. Also happens to be a handy search tool if needed. Please see examples below the table.
Filters
Name
Description
Values
count The maximum number of messages to purge; the limit. Defaults to 100 An integer
author Match messages sent by this author. Do not mention people. Discriminator (#0001) is optional, but recommended for accuracy. Nicknames will work. Username#0001
channel Match messages sent in this channel. Defaults to current channel #{channel_name}
contains Match messages that contain this text word or phrase
startswith Match messages that start with this text word or phrase
endswith Match messages that end with this text word or phrase
links Match messages that contain web URLs true
invites Match messages that server invites true
images Match messages that contain image attachments true
embeds Match messages that contain embeds, including embedded URLs true
mentions Match messages that contain @mentions true
bots Match messages that were sent by bot accounts true
attachments Match messages that contain attachments, such as images or files true
reactions Match messages that contain reactions true
external_messages Match messages that contain 3rd party embeds, such as Spotify invites or game invites through Discord (e.g. Fortnite) true
before Match messages that were sent before this date mm/dd/yyyy
after Match messages that were sent after this date mm/dd/yyyy
purge Include 'purge:true' to purge, exclude to pre-scan to see how many messages would be purged, if you included it true

This example isn't necessarily realistic, but includes every option, so you can follow this guide for your filters. Quotes are only required for multiple-word values, and again, all filters are optional

And if you find this daunting, you aren't alone. An interactive purge walkthrough is in the works.

.purge
count:5
author:Disguard#1890
contains:noob
"startswith:you are"
endswith:idiot
links:true
invites:true
images:true
embeds:true
mentions:true
bots:true
channel:#general
attachments:true
reactions:true
external_messages:true
before:4/27/19
after:1/1/19
purge:true

Here's a more realistic example - purging the most recent 20 messages in #bots sent by Mee6#4876 that contain image attachments:

.purge count:20 author:Mee6 channel:#bots images:true purge:true

And here's what a purge pre-scan looks like: (.purge channel:#general after:4/25/19)
