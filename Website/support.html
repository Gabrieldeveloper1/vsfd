Support command

Brings up an embed that allows you to start a Disguard support ticket

The Disguard Ticket System is a tool allowing server members to easily get in touch with Disguard's developer if an issue arises with the bot, or someone has a question. It does not support communications between server members and server admin, but if this is a request, perhaps it will in the future

You'll be able to manage your ticket / respond to the conversation either in DMs or in a server with essentially identical functionality

.support |{message}|

{message} - optional

The message to begin the support ticket with. If omitted, the bot will prompt for it when you create the ticket

Examples

.feedback How do you set up the logging channel?

Creates a support ticket with the message "How do you set up the logging channel?"
