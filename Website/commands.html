List of commands

Default perfix: . (period) - customizable through your server dashboards

Arguments surrounded by |pipestems| are optional

Click on the name of a command to learn more about it, and see usage details and examples

Command name Command description
help Shortcut to this page
support Create a support ticket through Disguard to get in direct contact with my developer if you have issues or questions about the bot
tickets View the support tickets you are a participant of
invite Pulls up bot invite link and invite link to the Disguard's official Discord server
pause {module} |{duration}|

unpause {module}		 Pause or unpause an active module (logging or antispam)
server Snapchat of basic server configuration (Logging and Antispam) - will be improved on in the future
info |{search term}| Intelligent search feature for server objects: Members, roles, channels, emojis, invites, etc.
birthday |{action}| View upcoming birthdays, your birthday information, another member's birthday information, or set your birthday/age
purge |{number}| Purge {number} messages. Omit {number} to enter interactive setup
lock {member} |{reason}|

unlock {member}		 Lock a member out of the server (prevent them from accessing channels), or unlock them and remove this restriction
mute {members} |{duration}| |{reason}|

unmute {members} |{reason}|		 Mutes a list of members, preventing them from speaking in channels, or unmute them and remove this restriction
kick {members} |{reason}| Kicks a list of members from the server
ban {users} |{reason}|

unban {users} |{reason}|		 Ban or unban a list of members from the server
history |{member}| |{module}| View the attribute history of {member} (yourself if omitted), such as username history, avatar history, custom status history, and more to come
