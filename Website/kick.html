Kick command

Kick a member from the server

Requires Kick Members permissions

.kick {members} |{reason}|

{members}

A list of members you wish to mute or unmute. Separate member inputs with a space, and each argument must exactly match (Case Sensitive) one of the following formats:

  • Member ID
  • Member Mention
  • Username#0000
  • Username
  • Nickname

{reason} - optional

Shows up in the audit log when kicking members

Examples

.kick Dyno rogue bot

Kicks Dyno from the server with the reason being displayed as "rogue bot"
