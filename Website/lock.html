Lock command

Lock or unlock members from a server

Locking a member removes their access to all channels, and unlocking them reverses it

Requires Manage Channels permissions because the bot uses channel permission overwrites to execute this command

.lock {member} |{reason}|

.unlock {member}

{member}

The member to lock or unlock. For the time being, must exactly match (Case Sensitive) one of the following formats:

  • Member ID
  • Member Mention
  • Username#0000
  • Username
  • Nickname

{reason} - optional

Only applies to the lock command

If provided, this will be DMd to the member upon being locked out

Examples

.lock Hijacker#4554

Lock Hijacker out of the server, removing their access to channels

.lock Troller Please change your profile picture to conform with our server rules and the Discord Terms of Service to be readmitted to the server

Lock Troller out of the server and let them know why

.unlock Troller

Restore server access to Troller
