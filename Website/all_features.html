All Features

If you want to learn more about the features Disguard offers, join the support server from the link bar on top and ask away!

Antispam

Your server is a place for its members, not raiders

  • Ability to flag messages with
    • Duplicated messages
    • Flood (posting too much too quickly)
    • Too many emoji
    • Too many member or role tags
    • Too many capital letters
    • Server invites
    • Web addresses
    • Attempted @everyone/@here tags
    • Certain types of attachments
    • Custom profane words
  • Exclude channels/roles/members from the antispam filter
  • Choose how many warnings members should have
  • Choose consequences for spamming
    • Nothing
    • Automute
    • Kick
    • Ban
    • Mute with custom role
  • Choose to send logs when filter reacts
  • Choose to delete the flagged message
  • Option to block members under a certain account age from joining your server
  • Option to ban members who repeatedly try to join your server in a short time span

Moderation

Designed with speed and flexibility in mind

  • Interactive purge command
    • Optional walkthrough menu allows for extreme precision with filters
  • Mute command
  • Lockout command
    • Prevents channel access for affected members
  • Kick command
  • Ban command
  • Disguard is gearing up for new moderation features in light of its public release!

Logging

Never miss any server changes

  • Message pin, edit, delete logs
    • Enabling message indexing allows message logs to work even with old messages
    • Message indexing allows ability to view message edit history
    • Enabling attachment storing allows ability see message attachments with deleted message logs
    • Enabling audit log reading displays who deleted or pinned messages
  • Channel create, edit, delete logs
    • Channel create logs display who has access to the channel
    • When permission overwrites were updated, you'll see the changes broken down
    • You'll see the channel's position in the list on all three log types
    • See deleted message count on channel deletion if message indexing is enabled
  • Role create, edit, delete logs
    • When applicable, permission changes will be displayed on logs
  • Member join, leave, update nickname, update role logs
    • Member join logs will display member and invite information
    • Member gain/lose role logs will display permission changes
  • User name, avatar, discriminator change logs
    • Avatar logs use permanent image storage so you can view media even months later
  • Emoji create, update, delete logs
  • Voice channel logs
    • Option to only enable moderator-enforced options or all options
    • Option to post activity summary upon a member ending their voice session
  • Miscallaneous logs
    • Ghost reaction logging
  • Enabling audit log reading displays the moderator behind an action
  • Option to repost a log upon its deletion if mods intend to hide their actions
  • Logs provide customizable features per category
    • Display only emojis, emojis and text, or only text in embed titles/descriptions
    • Use unicode emojis or custom emojis consistent with the Discord UI
    • Enable or disable media in embed author, thumbnail, and image fields
    • Enable or disable embed timestamp fields
    • Enable or disable logging of actions performed by bots
    • Log in embed form or plaintext form
    • Enable/Disable TTS logging
