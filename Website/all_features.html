All Features

For specific questions on features, click the 'join support server' link up top. All features are customizable. Some features aren't available yet.

Anti-spam

Automation level can be customized

  • Flag messages with
    • Duplicate/repeated content
    • Too many emoji
    • Too many member or role tags
    • Selfbot content
    • Too many capital letters
    • Discord.gg invites
    • Web addresses (http[s] URLs)
    • Attempted @everyone/@here tags
    • Spammed messages sent too quickly
    • Common or custom profanity
  • Exclude channels/roles/members from anti-spam at will
  • Choose to warn members, and how many if so
  • Set up a consequence for any offense
    • Do nothing, AutoMute, kick, ban, give custom role
  • Choose to allow consequences to be logged
  • Choose to delete the flagged message

Moderation

All commands have reciprocals (unmute, etc)

  • Quick commands to
    • Pin messages
    • Purge messages
    • Mute members
    • Kick members
    • Ban members
    • Softban members
      • Bans then immediately unbans members
    • Add role(s) to members
    • Lock server
      • Prevents members from texting/talking anywhere
    • Hide server
      • Prevents members from entering channels, & effects of lock
    • Lock channel
    • Hide channel
    • Warn members

Logging & Security

Keep guard of your server

  • Complete logging suite
    • Message: Edit, Delete
    • Channel: Create, Edit, Delete
    • Role: Create, Edit, Delete
    • Member: Join, Leave, Nickname, Role, Permissions
    • Emoji: Create, Edit, Delete
    • Audit log reading enables display of who changed what
    • Settings can be customized invididually or by group
    • Control embed color and channel
    • Optional advanced mode enables embed viewer capability
  • Get security rating for server or channel
    • Complex breakdown of things you may not consider
  • Automation, combined with anti-spam, secures server
  • Cybersecurity commands recommend improvements
