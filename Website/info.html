Info command

Search for and view summarized or advanced information of server data types

.info |{search term}|

{search term}

A search term or keyword. If omitted, the command will pull up information about your account.

Command searches through the following data types

  • Server
  • Channels
  • Roles
  • Server members
  • Emojis
  • Invites

Additionally, you may provide a keyword as a search term to quickly get to a summary view

  • 'Server' or 'guild' in search term: Pulls up server overview and statistics
  • 'Roles' in search term: Pulls up role list overview and statistics
  • 'Members' / 'people' / 'users' / 'bots' in search term: Pulls up member list overview
  • 'Channels' in search term: Pulls up channel list overview and statistics
  • 'Emoji' or 'emotes' in search term: Pulls up emoji list overview and statistics
  • 'Invites' in search term: Pulls up invite list overview and statistics
  • 'Bans' in search term: Pulls up ban list overview and statistics
  • 'Hardware' in search term: Pulls up hardware hosting information and bot statistics

Examples

.info general

Searches for objects with 'general' in the name

.info 558025201753784323

Searches for objects with this ID, which happens to be Disguard's

.info bans

Pulls up a list of banned members for the server (ban members permissions required)

.info xSGujjz

Searches for objects with 'xSGujjz' in the name, which happens to be an invite code in this case
