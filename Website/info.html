info command

Find information and statistics about many server data types

.info {search term}

{search term}

What you want to search for. Case insensitive; everything is converted to lowercase
Searches through the following items

  • Server name
  • Channels
  • Roles
  • Server members
  • Invites
  • Emojis

Keywords

  • 'Server' or 'Guild' in search term: Server overview and statistics available in results
  • 'Roles' in search term: Server roles overview and statistics available in results
  • 'Members'/'People'/'Users'/'Bots' in search term: Server members overview and statistics available in results
  • 'Channels' in search term: Server channels overview and statistics available in results
  • 'Emoji' or 'Emotes' in search term: Server emojis overview and statistics available in results
  • 'Invites' in search term: Server invites overview and statistics available in results
  • 'Bans' in search term: Server bans overview and statistics available in results
  • 'Dev'/'Owner'/'Master'/'Creator'/'Author'/'Disguard'/'Bot' in search term: Information about the bot and its owner

Examples

.info general

  • Searches for objects with 'general' in the name. Will probably return a list of a few channels

.info emojis

  • Searches for objects with 'emojis' in the name, and server emoji list will be included in results

.info a

  • Searches for anything with 'a' in the name. Please don't do this... it's spammy
