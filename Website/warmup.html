Warmup command

Configure the duration members must be in the server before being allowed to chat

This command mutes members when they join the server, then unmutes them once the designated amount of time has passed.

Requires Manage Roles and Manage Channels permissions because the bot uses these permissions to mute members

.warmup {duration}

{duration}

The new value to set. Format it with a number followed by a letter representing the units. For example, 5d would mean 5 days. Here are the acceptable values for letters representing units:

  • s: seconds
  • m: minutes
  • h: hours
  • d: days
  • w: weeks

Make sure the argument is one word so that the bot can properly do conversion.

Examples

.warmup 12h

Set the configuration so that members must be in the server for 12 hours before chatting

.warmup 0s

Disable the feature
