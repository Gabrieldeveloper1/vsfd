Disguard

An advanced logging, antispam, moderation, and cybersecurity bot working for you

If you're a manager or owner of a popular Discord server, you don't have to stress about making sure all the management and security settings are configured properly. Disguard specializes in providing features and interfaces to simplify the process of managing a server. From advanced logging features to highly customizable antispam filters to powerful moderation commands, Disguard manages the hard work so you can focus on interacting with your community.

Here are some highlights that Disguard offers:

  • Keep track of the activity in your server with highly informative logging features, including attachment logging
  • Ensure spammers don't ruin your server's experience with a customizable antispam suite
  • Moderation commands designed to act quickly: mute, quarantine, kick, or ban members in bulk
  • An information command to pull up any necessary information about server objects such as members, roles, or even invites
  • Bonus features such as useful utility features, Reddit feeds, a birthdays module, attribute history tracker, and more
  • Learn more about Disguard's features

Thanks to the feedback from Disguard's beta phase, the bot is on a path to be fully released in early 2022. Work is being done around the clock to ensure Disguard's launch provides innovative features to server moderators to improve their experience and enhance their abilties as moderators.
